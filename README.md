# 🛡️ SECURITY AUDITOR (v1.5.0)

## Strategic Security Auditing & Resilience Research Lab

**SECURITY AUDITOR** is an elite, autonomous intelligence platform designed for deep-spectrum security assessments, prompt resilience mapping, and systemic vulnerability discovery. It leverages a parallel swarm of agentic cores to validate organizational security boundaries and provide actionable hardening intelligence.

---

## 🚀 Key Capabilities

- **Strategic Simulation (Hybrid Vector):** Autonomous multi-step logic probing across web and internal environments to map potential exposure chains.
- **Resilience Boundary Validation:** Systematically probing LLM reasoning consistency and safety guardrail alignment.
- **Advanced Surface Mapping:** Recursive discovery of API surfaces, hidden administrative endpoints, and systemic misconfigurations.
- **Multimodal Security Verification:** Using AI vision to audit visual data flows and UI security consistency.
- **High-Fidelity Reporting:** Generates professional `Strategic Security Hardening Logs` with OWASP-aligned risk taxonomy.

---

## 🛠️ Technology Stack

- **Core:** Node.js / Bun
- **Intelligence:** Swarm Agentic Architecture (Support for OpenAI, Anthropic, Gemini, Kilocode)
- **Vision:** Integrated Multi-Model Vision Support
- **Interface:** High-Fidelity TUI (Terminal User Interface) & Real-time Web Dashboard

## 🚦 Getting Started

### 1. Installation

```bash
git clone https://github.com/JonusNattapong/AuditorAi.git
cd AuditorAi
npm install
```

### 2. Configuration

Create a `.env` file or use the built-in config manager:

```env
OPENAI_API_KEY=your_key_here
ANTHROPIC_API_KEY=your_key_here
# Optional: GOOGLE_SEARCH_KEY, KILOCODE_API_KEY
```

### 3. Launching the Lab

**TUI Mode (Professional Shell):**

```bash
npm run start
```

**Web Dashboard (Real-time Monitoring):**

```bash
npm run web
```
